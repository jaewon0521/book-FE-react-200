# FE-react-200

## 소개

교재 : 초보자를 위한 리액트 200제 입니다.

이 프로젝트는 react와 node.js express 프레임워크를 사용하여 REST API를 구현한 프로젝트입니다.

## 개발 환경

1. node.js v16.9.1

2. npm v7.21.1

3. MYSQL

4. window 10

5. chorme

## 프로젝트 실행 방법

프로젝트 경로에서 다음과 같이 명령어를 실행해주세요.

    npm install

```
npm run dev
```
